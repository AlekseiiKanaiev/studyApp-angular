Маршрутизация

Параметры маршрута

При определении ссылки на компонент, который использует параметры маршрута, все необходимые параметры заключаются в квадратные скобки

item 5

Данный подход работает для одной ссылки, но что если у нас несколько ссылок на один и тот же компонент, но с разными id?

item 5 item 8

Если мы сначала перейдем по первой ссылке, то компонент ItemComponent получит id=5. Если после этого мы перейдем по второй ссылке, ItemComponent будет продолжать хранить прежнее значение id = 5. Чтобы решить эту проблему, нам надо динамически изменять значение id в компоненте ItemComponent.

Параметры строки запроса

Кроме параметров маршрута в запросе могут передаваться параметры строки запроса. Например, в запросе http://localhost:3000/item?product=phone&price=200 часть product=phone&price=200 будет представлять параметры запроса - product и price. Такие параметры помещаются в конце адреса после вопросительного знака и разделяются амперсандом. Параметры строки запроса не влияют на определение маршрутов, и их количество произвольно.
